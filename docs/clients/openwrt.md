# OpenWrt

OpenWrt 适合需要在路由器侧统一接管设备流量的用户。路由器端配置比普通定制客户端更复杂，配置前请确认你了解当前路由器环境，并保留可恢复的管理入口。

## 下载地址

<a class="doc-button" href="https://down.zcjdgo.de/soft/artifact-openwrt.zip">查看 OpenWrt 安装说明</a>


## 适用场景

- 家中多台设备需要统一使用同一套代理配置。
- 电视、游戏机等设备不方便单独安装客户端。
- 你已经熟悉 OpenWrt 的基本管理方式。

## 配置节点

1. 登录 OpenWrt 管理后台。
2. 进入对应代理插件页面。
3. 找到节点、配置或订阅管理入口。
4. 按插件要求填写账号、节点配置或订阅信息。
5. 保存并同步配置。

## 开启代理

同步配置后，选择节点并启用服务。建议先只让一台测试设备通过路由器连接，确认正常后再扩大使用范围。

## 同步配置

如果节点不可用或列表为空，请先在插件中同步配置。必要时重启代理服务，但不要反复重启路由器。

## 常见问题

- 如果同步配置失败，请确认路由器本身可以访问网络。
- 如果只有部分设备可用，请检查分流规则和 DHCP 设置。
- 如果无法进入路由器后台，请使用保留的本地管理入口恢复配置。

## 无法登录或无法同步节点

如果客户端出现无法登录、无法同步节点，或提示 `Request failed with status code 500`，通常是因为其他代理软件未正常退出，导致系统仍处于代理模式。

建议先按下面方式处理：

1. 右键点击屏幕右下角托盘中的众筹加速器。
2. 选择「清除配置并重启」。
3. 如果没有该选项，说明当前版本较旧，请重新下载安装最新版。

![](https://pic.ybfl.xyz/i/2023/12/31/k6cvyy-1.jpg)

也可以手动检查系统代理：

1. 退出所有同类代理软件，包括本站客户端。
2. 打开浏览器设置，进入系统设置中的计算机代理设置。
3. 关闭代理设置中的所有代理开关。
4. 重新打开客户端并连接。

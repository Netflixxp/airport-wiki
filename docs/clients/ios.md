# iPhone / iPad

iPhone 和 iPad 当前使用 Shadowrocket。它不是本站定制客户端，因此需要按 iOS 客户端规则完成一键导入或手动配置。请尽量购买正版客户端，避免使用来源不明的共享账号。

!!! warning "安全提示"
    Shadowrocket 是付费软件，价格通常为 2.99 美元。近期共享 Apple ID 风险较高，可能导致账号异常或设备受影响。如需购买独享账号，可访问：`https://shop.zcjd.xyz/`

## 安装 Shadowrocket

1. 打开 iPhone 或 iPad 上的 App Store。
2. 点击右上角头像，下滑到底部并退出当前 Apple ID。
3. 登录购买后收到的 Apple ID。
4. 如果出现「Apple ID 安全」提示，选择「其他选项」，再选择「不升级」继续登录。

![](https://pic.ybfl.xyz/i/2023/02/23/yw9xwa-0.jpg)

可以通过下面链接直达 Shadowrocket 页面：

<a class="doc-button" href="https://apps.apple.com/us/app/shadowrocket/id932747118">打开 App Store 下载页</a>

如果无法直达，也可以在 App Store 搜索 `Shadowrocket` 后下载。

!!! tip "下载后请恢复自己的 Apple ID"
    下载完成后，请退出用于下载的 Apple ID，并重新登录你自己的 Apple ID。

## 一键导入订阅

请在要使用的 iPhone 或 iPad 上，用 Safari 浏览器打开本站和用户中心。一键导入按钮通常只在 Safari 中能正常唤起客户端。

<a class="doc-button" href="shadowrocket://add/sub://{{safeBase64SubscribeUrl}}?remark=INSNOW">导入到 Shadowrocket</a>

## 方式一：一键订阅

1. 使用 Safari 登录个人中心。
2. 在仪表盘或我的订阅页面找到一键订阅按钮。
3. 选择导入到 Shadowrocket。
4. 系统会跳转到 Shadowrocket，并自动填入节点订阅信息。

![](https://pic.ybfl.xyz/i/2023/02/22/10zhy4x-0.png)

导入完成后，客户端首页会显示节点和相关信息。

![](https://pic.ybfl.xyz/i/2023/02/22/11140tl-0.png)

进入配置页面，建议使用默认配置文件。

![](https://pic.ybfl.xyz/i/2023/02/22/10zi0t8-0.png)

进入设置页面，建议开启订阅的「打开时更新」和「自动后台更新」。

![](https://pic.ybfl.xyz/i/2023/02/22/10zhv4u-0.png)

## 方式二：手动订阅

1. 使用 Safari 登录个人中心。
2. 在仪表盘或我的订阅页面选择复制订阅地址。

![](https://pic.ybfl.xyz/i/2023/02/22/113lf8n-0.png)

3. 打开 Shadowrocket，点击右上角 `+`。

![](https://pic.ybfl.xyz/i/2023/02/22/113l3uk-0.png)

4. 类型选择 `Subscribe`。
5. 将订阅地址粘贴到 URL 栏。
6. 在备注区域输入分组名称。
7. 点击右上角完成保存。

![](https://pic.ybfl.xyz/i/2023/02/22/114vkht-0.png)

保存后，客户端会自动下载节点和相关信息。

![](https://pic.ybfl.xyz/i/2023/02/22/11140tl-0.png)

进入配置页面，建议使用默认配置文件。

![](https://pic.ybfl.xyz/i/2023/02/22/10zi0t8-0.png)

进入设置页面，建议开启订阅的「打开时更新」和「自动后台更新」。

![](https://pic.ybfl.xyz/i/2023/02/22/10zhv4u-0.png)

## 开启代理

回到首页，点击右上方系统总开关即可开启代理服务。

全局路由可以按需要调整：

- 「配置」：按配置文件规则自动分流，推荐大多数用户使用。
- 「代理」：所有连接都通过代理。
- 「直连」：所有连接都不通过代理。
- 「场景」：按不同网络环境自动切换代理模式。

## 常见问题

- 如果更新 App 时提示输入其他账号密码，通常是因为旧版本使用了不同 Apple ID 下载。卸载后使用当前账号重新安装即可。
- 如果一键导入无法跳转，请确认正在使用 Safari 浏览器。
- 如果节点为空，请重新复制订阅或重新点击一键导入。

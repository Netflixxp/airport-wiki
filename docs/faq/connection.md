# 连接问题

如果定制客户端已经登录，但仍然无法正常连接，可以按本页顺序排查。

## 先确认客户端状态

- 客户端是否已经连接到某个节点。
- 系统代理、增强模式或 VPN 开关是否已经开启。
- 客户端是否已经同步最新节点或配置。
- 当前节点是否显示超时或不可用。

## 更换节点测试

单个节点不可用不代表账号或客户端异常。建议先切换到其他地区或延迟更低的节点，再测试网页访问。

## 更换网络测试

如果所有节点都不可用，可以尝试切换网络：

- 从 Wi-Fi 切换到移动网络。
- 重启路由器后再试。
- 关闭其他代理、加速器或安全软件后再试。

## 仍然无法连接

请联系支持，并提供以下信息：

- 设备平台。
- 客户端名称。
- 报错截图。
- 当前网络类型。
- 是否可以正常登录客户端。

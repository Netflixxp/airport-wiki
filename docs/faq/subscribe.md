# 账号与节点

本站各平台优先提供定制客户端。定制客户端已经预先适配本站服务，安装后输入账号密码即可使用，通常不需要手动复制链接、导入地址或维护复杂配置。

## 流量消耗异常

如果你发现流量消耗明显异常，请先检查是否存在以下情况：

- 将订阅链接分享给了他人
- 在第三方网站进行了测速
- 客户端开启了自动选节点、故障转移等功能
- 误连接到了高倍率节点

虽然本站不限制终端数量，理论上你可以和朋友一起使用，但仍然建议不要分享你的订阅链接，也不要将订阅链接提交到第三方测速网站。

如果发生了以上情况，建议第一时间登录官网，重置你的订阅信息。

操作路径：

```text
官网 -> 个人中心 -> 页面最下方 -> 重置订阅信息
```

重置完成后请注意：

- 原有订阅链接会失效
- 原先已经导入的节点也会失效
- 只有使用新的订阅链接重新更新后，才能恢复正常使用

如果你使用的是需要手动导入订阅链接的客户端，例如 Shadowrocket、OpenWrt 或其他手动订阅客户端，请用新的订阅链接重新更新。

为了避免旧节点失效后更新失败，更新时建议先关闭代理模式，使用直连方式更新订阅。

## 什么是 2 倍率 / 10 倍率节点

倍率表示该节点的流量扣费比例。

- `1 倍率`：使用 `1GB`，扣除 `1GB`
- `2 倍率`：使用 `1GB`，扣除 `2GB`
- `10 倍率`：使用 `1GB`，扣除 `10GB`

目前部分高端 `IPLC` 节点会按 `10 倍率` 计算。例如你实际使用了 `1GB`，系统会记录为 `10GB` 流量消耗。通常节点名称会有明显标识，请使用前留意。

![10倍率节点示例](https://s2.loli.net/2022/11/07/E6CsZ3oycMPrxOQ.png)

另外，部分客户端有自动选节点或故障转移功能。由于 `IPLC` 节点通常延迟更低，自动选择时很容易优先选中这些高倍率节点。

如果你没有特殊需求，建议关闭客户端的自动选节点功能。

![自动选节点示例](https://s2.loli.net/2022/11/07/cfbgrGiXTd2wkns.png)

如果你在 10 倍率节点的流量记录里只看到几十 KB 的消耗，通常是客户端测速或测试连接时产生的流量，这种情况一般可以忽略。

## 登录后没有节点

请先确认：

- 账号和密码输入正确。
- 套餐仍在有效期内。
- 当前网络可以打开官网或用户中心。
- 客户端是从本站提供的入口下载。

如果仍然没有节点，请退出客户端后重新登录。

## 节点列表没有更新

定制客户端一般会自动同步节点和配置。如果节点列表长时间没有变化，可以按下面顺序处理：

1. 退出客户端并重新打开。
2. 在客户端内查找「刷新」「同步」「更新配置」等入口。
3. 切换网络后重新登录。
4. 确认官网账号状态正常。

## 仍然无法使用

请联系支持，并附上以下信息：

- 设备平台。
- 客户端名称。
- 账号或订单信息。
- 问题截图或报错文字。

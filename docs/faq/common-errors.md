# 常见报错

不同客户端的提示文字可能略有差异，但排查方向基本相同。建议先确认账号可以登录，再检查系统代理和节点状态。

## 节点全部超时

可能原因：

- 当前网络对连接有限制。
- 客户端配置长时间未同步。
- 客户端没有开启系统代理或 VPN 权限。
- 节点临时维护。

建议先重新登录或同步配置，再切换其他节点测试。

## 无法打开网页

请检查：

- 客户端是否已经连接。
- 系统代理或 VPN 开关是否已开启。
- 浏览器是否设置了单独代理。
- 是否只是在某个网站打不开。

## 连接后国内网站变慢

可以尝试切换到规则模式，或选择延迟更低的节点。如果客户端支持配置同步，请先同步配置后再测试。

## 证书或权限提示

部分客户端首次启动时会请求 VPN、网络扩展或系统代理权限。请按照系统提示允许，否则客户端可能无法接管流量。

# Windows 客户端

请选择一个 Windows 定制客户端进入教程。定制客户端已经按本站服务预先适配，安装后输入账号密码即可使用，可避免手动复制链接、导入地址和维护复杂配置等对新用户不友好的操作。

!!! info "如何选择"
    - 首次使用，建议优先选择 **客户端A**。
    - 如果你更熟悉旧版界面，可以继续使用 **客户端B**。
    - 两个客户端只需安装一个，不建议同时开启多个系统代理客户端。
